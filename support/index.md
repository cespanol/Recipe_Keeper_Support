# Recipe Keeper Support

For help, open a [Recipe Keeper support issue](https://github.com/cespanol/Recipe_Keeper_Support/issues/new?template=support.yml). GitHub issues are public, so do not attach recipe content, source links, photos, clipboard contents, filenames or paths, iCloud identifiers, or personal account information.

Include only:

- App version and build number from Settings
- iOS/iPadOS version and device class, such as “iPhone” or “iPad”
- The storage/account status shown by Recipe Keeper
- The content-free diagnostic event code, if one is shown
- A short sequence of actions and what you expected to happen

## Safe first steps

1. Preserve any existing Recipe Keeper library backup.
2. Confirm the device has free storage and an internet connection.
3. If iCloud behavior is involved, check that iCloud Drive is enabled and that the same Apple Account is signed in.
4. Restart Recipe Keeper and retry once.
5. Do not delete or reinstall the app as a first troubleshooting step; local data may not be recoverable unless same-account iCloud recovery or an explicit backup has already been verified.

Security or privacy reports should include only the minimum information needed to reproduce the issue. If a report cannot be shared safely in public, open a minimal issue requesting a private contact route without including sensitive details.
